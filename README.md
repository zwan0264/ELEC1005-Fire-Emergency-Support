# Fire Emergency Support

This repository contains the source files and documentation for our ELEC1005 Assignment 2 Power Apps prototype.

## Project Overview
Fire Emergency Support is a mobile emergency response prototype developed from our Assignment 1 design.

The system supports:
- Civilian emergency help requests
- Responder incident management

## Core Features
- Role-based home screen
- Civilian request flow
- Responder dashboard
- Incident detail page
- Status updates
- SharePoint / Microsoft Lists backend
